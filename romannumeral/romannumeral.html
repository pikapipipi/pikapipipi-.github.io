Roman Numeral Converter


Roman numerals are based on seven symbols and can be written using various combinations to represent numerals, for example:

1984
= (1×1000) + (1×500 + 4×100) + (1×50 + 3×10) + (4×1)
= (M) + (D + CCCC) + (L + XXX) + (IIII)
= MDCCCCLXXXIIII
 OR
= (1×1000) + (1×1000 - 1×100) + (1×50 + 3×10) + (5 - 1)
= (M) + (-C + M) + (L + XXX) + (-V + I)
= MCMLXXXIV
